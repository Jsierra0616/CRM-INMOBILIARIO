# CRM INMOBILIARIO

Proyecto CRM para gestión de propiedades inmobiliarias.

## Tecnologías
Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express

## Instalación del backend

1. Entrar a la carpeta backend

cd backend

2. Instalar dependencias

npm install

3. Crear archivo .env basado en .env.example

4. Ejecutar servidor

node src/server.js
